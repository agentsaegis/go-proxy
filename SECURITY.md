# Security Policy

## Reporting a vulnerability

Do not open a public issue for security vulnerabilities. Instead, email **security@agentsaegis.com** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and provide an initial assessment within 5 business days.

## Scope

This policy covers the AgentsAegis proxy (`go-proxy`). For issues with the AgentsAegis dashboard, contact the same email.
